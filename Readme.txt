TO RUN THE APP:
First activate the virtual environment:  .\\env\Scripts\activate.bat
Then install all requirements: pip install -r requirements.txt
To run: python app.py

When the app runs, a welcome page will be shown which will ask for login and signup
User have to click signup to create a new account.
user can choose to be admin/store manager to manage groceries while signup
Then user can start after logging in.
regular User can add products to cart
User can also search for products
admin can manage categories and products
